# 📄 CV Generator Website

A dynamic CV Generator website built with PHP and JavaScript.

## 🛠️ Technologies Used
- PHP (Backend)
- JavaScript (Frontend Logic)
- HTML/CSS (Design)

## ✅ Features
- User fills a simple form
- CV automatically generates in real-time
- Download CV as PDF
- Clean and user-friendly interface
